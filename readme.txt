# Python_Formulaire

This is a simple Flask web application for managing student information. Users can input details about a student, and the server will display the student's information along with additional details.

## Project Structure

- `app.py`: Contains the main Flask application.
- `Classes.py`: Defines the `Etudiant` class representing a student.
- `index.html`: HTML template for the web form and displaying student information.
- `template.css`: CSS styling for the HTML elements.
